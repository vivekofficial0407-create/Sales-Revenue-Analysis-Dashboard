# Sales & Revenue Analysis Dashboard

## Project Overview
This dashboard was developed using Microsoft Power BI to analyze sales performance, profit trends, product performance, and country-wise sales distribution.

## Key Metrics
- Total Sales: 118.73M
- Total Profit: 16.89M

## Key Insights
- Top Performing Product: Paseo
- Top Performing Country: United States
- Highest Sales Month: October

## Features
- KPI Cards
- Product-wise Sales Analysis
- Country-wise Sales Analysis
- Monthly Sales Trend
- Interactive Country Filter

## Tools Used
- Microsoft Power BI
